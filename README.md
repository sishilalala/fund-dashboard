# Fund Dashboard

## Live Site

**https://shihtalks.github.io/fund-dashboard/**

Daily Chinese mutual fund screening dashboard with:
- A-share market overview (indices, industry heatmap, fund flow)
- Fund rankings by category (Top 50, equity, hybrid, bond, index, QDII, FOF)
- Multi-period performance (daily, weekly, 1/3/6 month, YTD)
- Industry news from authoritative Chinese financial media

Data auto-updates every trading day at 16:30 Beijing time via GitHub Actions.
